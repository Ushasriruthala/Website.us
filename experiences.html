Cloud Computing Intern

Project : Live streaming of RTMP Cameras using Amazon Web Services

I recently completed a two-month internship in Brihaspathi Technologies Private Limited, Hyderabad focused on cloud computing with a specific emphasis on Amazon Web Services (AWS). During this internship, I gained hands-on experience in various aspects of cloud technologies, including setting up virtual servers, managing data storage, and configuring networking solutions on the AWS platform. One of the highlights of my internship was a project where I worked on designing and implementing a solution for streaming videos from multiple cameras to the cloud using AWS services.

For the project, I collaborated with a team to architect a scalable and efficient solution to capture video feeds from diverse sources and stream them to the cloud for processing and analysis. I was responsible for setting up AWS services such as Amazon Elemental Medialive and Amazon Elemental Package for the video streams.

Through this internship, I not only enhanced my technical skills in cloud computing and AWS, but I also developed a deep understanding of video streaming.
